== Description ==

This simply shows an IP address on a specific page or widget by simply adding in a tag. Your IP address also shows on your admin dashboard.

== Installation ==

You can find, download and install 'Show IP address' directly from the **Plugins** section in WordPress.

If you want to install manually, download and unzip the search-meter.zip file and upload to the `/wp-content/plugins/show-ip-address` directory. Then activate the plugin through the **Plugins** section in WordPress.

= Widgets

This simply show an IP address on a specific page or widget by simply adding in this tag [show_ip].

= Template Tags =

[show_ip] use this tag on your pages to show your IP address.

== Frequently Asked Questions ==

Q). Will this work on any WordPress installation?
